# Example Playlist

This is an example playlist folder. Add your music files here!

## How to Use

1. Add your audio files (MP3, M4A, WAV, OGG, FLAC) to this folder
2. The bot will automatically detect and load them
3. Files are played in alphabetical order

## Supported Formats

- `.mp3` - MP3 audio
- `.m4a` - AAC/M4A audio
- `.wav` - WAV audio
- `.ogg` - OGG Vorbis
- `.flac` - FLAC lossless
- `.opus` - Opus codec
- `.webm` - WebM audio

## Tips

- Name your files with numbers for specific ordering (e.g., `01-song.mp3`, `02-song.mp3`)
- Keep file names simple (avoid special characters)
- Organize different playlists in separate folders
